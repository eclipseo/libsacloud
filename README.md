# libsacloud

Library for SakuraCloud API

## APIs

- [x] `Archive`: アーカイブ
- [x] `AuthStatus`: 認証情報
- [x] `AutoBackup`: 自動バックアップ
- [x] `Bill`: 請求
- [x] `Bridge`: ブリッジ
- [x] `Coupon`: クーポン
- [x] `CDROM`: ISOイメージ
- [x] `Database`: データベースアプライアンス
- [x] `Disk`: ディスク
- [x] `DNS`: DNS
- [x] `GSLB`: GSLB
- [x] `Icon`: アイコン
- [x] `Interface`: インターフェース
- [ ] `Internet`: スイッチ+ルータ(残: IPv6関連)
- [ ] `IPv4Addr/IPv6Addr`: IPv4/v6アドレス
- [x] `License`: ライセンス
- [x] `LoadBalancer`: ロードバランサ
- [ ] `MobileGateway`: モバイルゲートウェイ 
- [ ] `NewsFeed`: ニュースフィード(メンテナンス情報)
- [x] `NFS`: NFS
- [x] `Note`: スタートアップスクリプト
- [x] `PacketFilter`: パケットフィルタ
- [x] `PrivateHost`: 専有ホスト
- [ ] `ProductXXX`: 商品関連(productXXX)
- [ ] `Price`: 価格表(public/price)
- [x] `ProxyLB`: エンハンスドロードバランサ
- [x] `Region`: リージョン
- [x] `SimpleMonitor`: シンプル監視
- [ ] `SIM`: SIM
- [x] `SSHKey`: 公開鍵
- [x] `Subnet`: サブネット
- [x] `Switch`: スイッチ
- [x] `VPCRouter`: VPCルータ
- [x] `Zone`: ゾーン
- [ ] `WebAccel`: ウェブアクセラレータ
